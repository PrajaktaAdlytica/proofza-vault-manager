# Proofexa Website Assets

These files are ready for review:

- `proofexa-logo.svg` — horizontal logo lockup for the website.
- `proofexa-icon.svg` — standalone app/profile mark.
- `proofexa-favicon.svg` — favicon source.

Brand idea: continuous audit evidence operations. The document capsule represents the evidence vault, the stacked bars represent files/screenshots/policies/tickets, the green control tile represents verified audit readiness, and the small provenance nodes show evidence being pulled from connected systems.
